# Supermarket Sales Analysis and Prediction

**Self-taught Data Analysis & Machine Learning Project for AI/ML Job Applications**

## 📋 项目概述
本项目对超市杂货店销售数据进行端到端的探索性数据分析（EDA）和销售预测，旨在发现销售趋势、顾客行为洞察，并构建模型预测未来销售量，帮助零售业务优化决策。

## 🛠️ 技术栈
- **数据处理**：Pandas, NumPy
- **可视化**：Matplotlib, Seaborn
- **机器学习**：TensorFlow / Keras / PyTorch（用于销售预测模型）
- **AI辅助开发**：Gemini （代码生成、异常检测和洞察总结）
- **环境**：Google Colab / Jupyter Notebook

## 📊 主要工作与成果
- 使用Pandas和NumPy清洗数据（处理缺失值、异常值、类别编码）
- 通过Matplotlib和Seaborn创建多种可视化：销售趋势、产品类别表现、顾客类型分析、季节/时间模式等
- 识别关键业务洞察（如高销量产品、促销效果、潜在收入增长机会）
- 构建销售预测模型，并评估性能 mae
- 利用Gemini加速自动化分析脚本生成，减少手动工作时间

## 📥 数据来源
- 数据集来自Kaggle（Supermarket Sales / Supermart Grocery Sales 等零售数据集）
- 下载链接：[Kaggle Supermarket Sales Dataset](https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales) 

## 🚀 如何运行
1. 打开 `supermarket_sales_analysis_prediction.ipynb`
2. 在Google Colab中运行所有单元格
3. 数据集CSV文件可直接下载后放入 `data/` 文件夹

## 📌 项目意义
通过这个项目，我练习了从原始数据到业务洞察与预测的全流程，强化了数据分析和机器学习能力。非常适合应用于零售、电商等行业的AI解决方案。
