# Supermarket Sales Analysis and Prediction

**Self-taught Data Analysis & Machine Learning Project for AI/ML Job Applications**

## 📋 Project Overview
Performed end-to-end Exploratory Data Analysis (EDA) and built a sales prediction model on supermarket grocery sales data. The project uncovers sales trends, customer behavior, and provides actionable business insights for retail optimization.

## ✨ Highlights
- Conducted full EDA and sales trend analysis on real retail data
- Built a predictive model with evaluation metrics (MAE)
- Identified key drivers of sales (product categories, promotions, seasonal patterns)
- Used Gemini to accelerate analysis scripts and anomaly detection

## 🛠️ Tech Stack
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn
- **Machine Learning**: TensorFlow / Keras / PyTorch (for sales prediction)
- **AI Assistance**: Gemini (code generation, anomaly detection, insights)
- **Environment**: Google Colab / Jupyter Notebook

## 📊 Key Work & Results
- Cleaned data (missing values, outliers, categorical encoding) using Pandas and NumPy
- Created rich visualizations: sales trends, top products, customer segmentation, and seasonal patterns with Matplotlib/Seaborn
- Identified business insights such as high-performing categories and promotion effectiveness
- Developed a sales forecasting model and evaluated performance (MAE / other metrics)

## 📥 Dataset
- Source: Kaggle Supermarket Sales Dataset
- Download Link: [https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales](https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales)
- Place the CSV file in a `data/` folder before running

## 🚀 How to Run
1. Open `超市杂货店的数据分析与机器学习2.ipynb`
2. Run all cells in Google Colab
3. Download the dataset from Kaggle and put it in the `data/` folder

## 📌 Project Significance
This project helped me master the full workflow from raw data to business insights and predictive modeling. It is highly relevant to retail, e-commerce, and data-driven decision-making roles in AI/ML.
